📝 Summary

  • Static: Default. Normal flow. No positioning.
  • Relative: Offset from normal position. Space reserved.
  • Absolute: Removed from flow. Positioned relative to parent.
  • Fixed: Removed from flow. Positioned relative to viewport.
  • Sticky: Relative + Fixed hybrid. Sticks when scrolling.

🔑 Key Rule

Absolute children need position: relative parent!
