1. Translate (Move)

Move elements without affecting layout

translate(50px, 30px)
transform: translate(50px, 30px);

Move right 50px, down 30px

2. Rotate

Rotate around center point

rotate(45deg)
transform: rotate(45deg);

3. Scale

Resize element (1.3 = 130% size)

scale(1.3)
transform: scale(1.3);

scale(2) = double size, scale(0.5) = half size

4. Skew

Slant element along X and Y axes

skew(10deg, 5deg)
transform: skew(10deg, 5deg);

5. Multiple Transforms

Combine multiple transforms (space-separated)

📝 Transform Functions

Function Description Example
translate() Move element translate(50px, 20px)
translateX() / translateY() Move on single axis translateX(100px)
rotate() Rotate element rotate(45deg)
rotateX() / rotateY() 3D rotation rotateY(180deg)
scale() Resize element scale(1.5) or scale(2, 0.5)
scaleX() / scaleY() Scale on single axis scaleX(2)
skew() Slant element skew(10deg, 5deg)

⚡ Why Use Transform?

  • Performance: Transform uses GPU (60fps animations)
  • No Reflow: Doesn't affect document layout
  • Smooth: Better for animations than changing width/height
  • 3D Effects: Can create depth with perspective

🎯 Best Practices

  • Always use transform for animations (not top/left/margin)
  • Combine with transition for smooth effects
  • Use transform-origin to control pivot point
  • Add will-change: transform; for complex animations
  • For 3D, use transform-style: preserve-3d; on parent

💡 Common Patterns

/* Lift on hover */
.card:hover {
  transform: translateY(-5px);
}

/* Flip card */
.card:hover {
  transform: rotateY(180deg);
}

/* Zoom in */
.image:hover {
  transform: scale(1.1);
}

/* Pulse effect */
@keyframes pulse {
  0%, 100% { transform: scale(1); }
  50% { transform: scale(1.1); }
}
