📝 Transition Syntax

/* Shorthand */
transition: property duration timing-function delay;

/* Examples */
transition: all 0.3s ease;
transition: background-color 0.5s ease-in-out 0.1s;
transition: transform 0.3s cubic-bezier(0.175, 0.885, 0.32, 1.275);

/* Multiple properties */
transition: 
  background 0.3s ease,
  transform 0.5s ease,
  box-shadow 0.3s ease;

🎯 Best Practices

  • Duration: 0.2s-0.3s for UI interactions, 0.5s-1s for larger movements
  • Timing: ease-out for entrances, ease-in for exits
  • Performance: Transition transform & opacity (GPU accelerated)
  • Avoid: Transitioning layout properties (width, height, margin) - use transform instead

⚡ Common Timing Functions

  • linear: Constant speed (robotic)
  • ease: Slow start, fast middle, slow end (default, natural)
  • ease-in: Slow start, fast end (falling)
  • ease-out: Fast start, slow end (best for UI)
  • ease-in-out: Slow start and end
  • cubic-bezier(): Custom curves (advanced)
