💾 Demo 03: LocalStorage

setItem, getItem, removeItem, clear — saving data in the browser

1. The 4 Methods

localStorage.setItem('key', 'value'); // SAVE localStorage.getItem('key'); // GET (or null) localStorage.removeItem('key'); // DELETE one localStorage.clear(); // DELETE ALL

3. Practical Example: Remember User Theme

// Save theme preference localStorage.setItem('theme', 'dark'); // Load on page start const theme = localStorage.getItem('theme') || 'light'; document.body.className = theme;
