🔒 Demo 02: Form Validation

Validating user input, showing errors, real-time feedback

1. Basic Validation

function validateName(name) { if (name.trim() === '') return 'Name is required!'; if (name.length < 2) return 'Name must be at least 2 characters!'; return ''; // Empty string = valid }
✅ Form submitted successfully!

2. Real-Time Validation

input.addEventListener('input', function() { if (input.value.length >= 3) { input.classList.add('valid'); input.classList.remove('invalid'); errorDiv.classList.remove('show'); } else { input.classList.add('invalid'); errorDiv.textContent = 'Too short!'; errorDiv.classList.add('show'); } });
✅ Account created!
💡 Pattern: Validate on every keystroke (input event) AND on submit. Users get instant feedback while typing!
