🐛 Demo 06: Console & Debugging

👉 Open your console (F12) to see all the outputs!

1. console.log() - Basic Logging

console.log("Hello, World!");
console.log("Number:", 42);
console.log("Multiple values:", 1, 2, 3);

const user = { name: "Alice", age: 25 };
console.log("User object:", user);
✅ Use console.log() to check variable values while coding!

2. console.error() & console.warn()

console.error("This is an error!");
console.warn("This is a warning!");
console.info("This is info");

Check your console to see different colored outputs!

3. console.table() - Display Arrays/Objects

const users = [
  { name: "Alice", age: 25, city: "NYC" },
 & { name: "Bob", age: 30, city: "LA" },
  { name: "Charlie", age: 35, city: "Chicago" }
];

console.table(users);
✅ console.table() makes arrays of objects easy to read!

4. console.time() - Measure Performance

console.time("Loop Timer");

for (let i = 0; i < 1000000; i++) {
  // Do something
}

console.timeEnd("Loop Timer"); // Shows elapsed time

5. console.group() - Organize Logs

console.group("User Details");
console.log("Name: Alice");
console.log("Age: 25");
console.log("City: NYC");
console.groupEnd();

6. Debugging Techniques

Check Variable Values:

function calculateTotal(price, quantity) {
  console.log("Price:", price);
  console.log("Quantity:", quantity);
  
  const total = price * quantity;
  console.log("Total:", total);
  
  return total;
}

Check Function Flow:

function processOrder() {
  console.log("1. Starting order");
  validateItems();
  console.log("2. Items validated");
  calculateTotal();
  console.log("3. Total calculated");
}

debugger Statement:

function complexCalculation(x) {
  let result = x * 2;
  debugger; // Execution pauses here
  result += 10;
  return result;
}
⚠️ Remove debugger statements before deploying to production!

7. Common Debugging Workflow

  1. See error in console - Read error message carefully
  2. Check line number - Go to the line mentioned in error
  3. Add console.log() - Check variable values before error
  4. Fix and test - Make changes and refresh page
  5. Remove logs - Clean up console.log() statements

8. DevTools Tips

  • F12 - Open DevTools
  • Ctrl+Shift+C - Element inspector
  • Ctrl+L - Clear console
  • $0 - Reference selected element
  • ↑/↓ arrows - Navigate command history
