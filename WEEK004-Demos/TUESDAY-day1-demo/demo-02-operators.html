2. Comparison Operators

Operator Name Example Result
== Equal to (loose) 5 == "5"
=== Equal to (strict) 5 === "5"
!= Not equal (loose) 5 != "6"
!== Not equal (strict) 5 !== "5"
> Greater than 10 > 5
< Less than 3 < 8
>= Greater or equal 5 >= 5
<= Less or equal 4 <= 10
⚠️ Always use === and !==
Loose equality (==) does type coercion which can cause bugs!
Example: 0 == false is true, but 0 === false is false
