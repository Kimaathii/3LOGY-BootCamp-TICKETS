## TEACHING STRATEGY

### Demo Sequence:
Each demo includes:
- Live code examples in browser console
- Visual output displayed on page
- Common mistakes highlighted
- Practice exercises

### Interactive Testing:
- Students open browser console (F12)
- Type code along with instructor
- See results immediately
- Experiment with variations

## KEY CONCEPTS COVERED

### Demo 1: Variables & Data Types
- `var` vs `let` vs `const`
- Primitive types: string, number, boolean, null, undefined
- Template literals
- `typeof` operator

### Demo 2 Operators
- Arithmetic: +, -, *, /, %,  **
- Comparison: ==, ===, !=, !==, <, >, <=, >=
- Logical: &&, ||, !
- Assignment: =, +=, -=, etc.

### Demo 3: Functions
- Function declarations
- Function expressions
- Arrow functions
- Parameters and return values
- Scope basics

### Demo 4: Control Flow
- if/else statements
- else if chains
- switch statements
- Ternary operator (? :)

### Demo 5: Loops
- for loops
- while loops
- do...while loops
- for...of with arrays
- break and continue

### Demo 6: Console & Debugging
- console.log, console.error, console.warn
- console.table for arrays/objects
- Breakpoints in DevTools
- debugger statement
