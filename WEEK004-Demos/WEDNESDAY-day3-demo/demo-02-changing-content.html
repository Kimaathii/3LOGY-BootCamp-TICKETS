✏️ Demo 02: Changing Content

textContent, innerHTML, attributes, styles, classList

1. textContent vs innerHTML

// textContent — plain text only (SAFE!) element.textContent = "Hello World"; // innerHTML — can include HTML tags (be careful!) element.innerHTML = "<strong>Bold</strong> and <em>italic</em>";
Original text — click buttons to change me!

2. Changing Styles Directly

element.style.backgroundColor = 'blue'; element.style.fontSize = '24px'; element.style.borderRadius = '12px'; // CSS hyphens → camelCase in JS!
Style me!

3. classList — Toggle CSS Classes

element.classList.add('active'); element.classList.remove('active'); element.classList.toggle('active'); // Add if missing, remove if present element.classList.contains('active'); // true/false
Click buttons to toggle classes on me!
Current classes: demo-box
✅ Best Practice: Use classList instead of direct style changes. Define styles in CSS, toggle classes with JS!
