2. Form Submit — e.preventDefault()

form.addEventListener('submit', function(e) { e.preventDefault(); // ⚠️ CRITICAL! Stops page from reloading const value = input.value; // Process the value... });
⚠️ Without e.preventDefault(): The page reloads on form submit and you lose ALL your JavaScript data! Always include it.
