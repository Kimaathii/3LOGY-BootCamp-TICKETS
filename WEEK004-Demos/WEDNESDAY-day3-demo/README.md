# Wednesday Demos - DOM Manipulation

**Session:** Virtual Class (Wednesday 6:00-9:00 PM)  
**Topic:** Selecting Elements, Changing Content, Event Handling

---

## DEMO FILES

1. **demo-01-selecting-elements.html** - getElementById, querySelector, querySelectorAll
2. **demo-02-changing-content.html** - textContent, innerHTML, attributes, styles
3. **demo-03-event-handling.html** - click, input, submit events
4. **demo-04-creating-elements.html** - createElement, appendChild, removeChild
5. **demo-05-todo-app.html** - Complete interactive To-Do list example

---

## KEY CONCEPTS COVERED

### Demo 1: Selecting Elements
- `document.getElementById('id')`
- `document.querySelector('.class')`
- `document.querySelectorAll('selector')`
- Difference between querySelector and querySelectorAll

### Demo 2: Changing Content
- `element.textContent` - plain text
- `element.innerHTML` - HTML content
- `element.setAttribute()`
- `element.style.property`
- `element.classList.add/remove/toggle()`

### Demo 3: Event Handling
- `addEventListener('click', function)`
- Event object (e.target, e.preventDefault())
- Input events
- Form submit events

### Demo 4: Creating Elements
- `document.createElement('tag')`
- `parent.appendChild(child)`
- `parent.removeChild(child)`
- Building dynamic lists

### Demo 5: Complete App
- Full To-Do list application
- Add tasks
- Mark as complete
- Delete tasks
- Local workflow demonstration
