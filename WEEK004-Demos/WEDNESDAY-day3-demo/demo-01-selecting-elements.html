🔍 Demo 01: Selecting Elements

Finding HTML elements with getElementById, querySelector, querySelectorAll

1. getElementById

const title = document.getElementById('unique-title'); // Returns ONE element (or null if not found)

2. querySelector — First Match Only

document.querySelector('#myId'); // By ID document.querySelector('.myClass'); // By class document.querySelector('h1'); // By tag document.querySelector('.parent .child'); // Nested
Card 1 (class="card-item")
Card 2 (class="card-item")
Card 3 (class="card-item")
Click the button above

3. querySelectorAll — All Matches

const allCards = document.querySelectorAll('.card-item'); // Returns a NodeList (like an array) console.log(allCards.length); // 3 allCards.forEach(function(card) { card.style.backgroundColor = 'green'; });
