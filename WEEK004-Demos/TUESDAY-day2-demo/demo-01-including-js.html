🎯 Demo 01: Including JavaScript in HTML

Three ways to add JavaScript to your web pages

👉 Open your browser console (F12) and follow along!
This demo shows how JavaScript connects to HTML pages.

1. Inline JavaScript (onclick attribute)

The simplest way — add JS directly in HTML attributes:

<button onclick="alert('Hello from inline JS!')">Click Me</button>

Try it:

⚠️ Don't use inline JS in real projects:
It mixes HTML and JavaScript, making code hard to maintain. We're showing it only so you understand it when you see it.

2. Internal JavaScript (<script> tag)

Write JavaScript inside a <script> tag within your HTML file:

<body> <h1 id="title">Hello</h1> <!-- JavaScript goes here, BEFORE closing </body> --> <script> const title = document.getElementById('title'); title.textContent = 'Hello from Internal JS!'; </script> </body>

This text will change when you click the button.

Console Output:
✅ Best Practice:
Place your <script> tag just before the closing </body> tag. This ensures all HTML elements are loaded before your JS runs.

3. External JavaScript (separate .js file)

The recommended approach — keep JS in a separate file:

/* File: script.js */ const heading = document.getElementById('main-heading'); heading.textContent = 'Loaded from external file!'; console.log('External JS loaded successfully!');
<!-- In your HTML file: --> <body> <h1 id="main-heading">Hello</h1> <!-- Link to external JS file --> <script src="script.js"></script> </body>

This simulates loading from an external file.

✅ Why external JS?
  • Separates structure (HTML) from behavior (JS)
  • Browser can cache the JS file (faster loading)
  • Easier to maintain and debug
  • Can reuse the same JS file across multiple pages

4. Where to Place Your Script Tag?

<!-- ❌ BAD: Script in <head> without defer --> <head> <script src="script.js"></script> </head> /* Problem: HTML elements don't exist yet when JS runs! */ <!-- ✅ GOOD: Script before closing </body> --> <body> <h1>My Page</h1> <script src="script.js"></script> </body> <!-- ✅ ALSO GOOD: defer attribute --> <head> <script src="script.js" defer></script> </head> /* defer = load JS but wait until HTML is ready */

5. 🎯 Practice Exercises

Try these in your own HTML file:

  1. Create an HTML file with a <script> tag that logs "Hello World" to the console.
  2. Create a separate script.js file and link it to your HTML with <script src="script.js">.
  3. Add a button with an onclick that changes a paragraph's text.
  4. Try placing your <script> in the <head> — what happens? Now add defer.
💡 Key Takeaway:
For this course: put <script> before </body> — it's the simplest and most reliable approach!
