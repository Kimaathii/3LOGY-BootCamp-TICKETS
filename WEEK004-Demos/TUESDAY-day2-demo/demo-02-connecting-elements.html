🎯 Demo 02: Connecting JS to Page Elements

Finding HTML elements and reading or changing their content

👉 Open your browser console (F12) and follow along!
Watch how JavaScript finds and modifies HTML elements on the page.

1. Finding Elements by ID

// HTML: <p id="greeting">Hello</p> const element = document.getElementById('greeting'); console.log(element); // The entire <p> element console.log(element.textContent); // "Hello"

Hello, I'm a paragraph with id="greeting"

2. Finding Elements with querySelector

// By ID (use #) const byId = document.querySelector('#myId'); // By class (use .) const byClass = document.querySelector('.myClass'); // By tag name const byTag = document.querySelector('h1'); // querySelector returns the FIRST match only

First paragraph with class "highlight-me"

Second paragraph with class "highlight-me"

Third paragraph with class "highlight-me"

Output:
✅ Best Practice:
Use querySelector for single elements and querySelectorAll when you need multiple. They use CSS selector syntax (same as CSS!).

3. Reading and Changing Text

const heading = document.querySelector('#myHeading'); // READ text console.log(heading.textContent); // "Old Title" // CHANGE text heading.textContent = "New Title!"; // CHANGE HTML (can include tags) heading.innerHTML = "<em>Italic Title</em>";

Original Title — Click buttons to change me!

Output:
⚠️ textContent vs innerHTML:
  • textContent — plain text only (safe, prevents XSS)
  • innerHTML — can include HTML tags (be careful with user input!)

4. Changing Styles with JavaScript

const box = document.querySelector('#myBox'); // Change individual styles box.style.backgroundColor = '#3498db'; box.style.color = 'white'; box.style.fontSize = '20px'; box.style.padding = '20px'; box.style.borderRadius = '10px'; // ⚠️ Note: CSS "background-color" becomes "backgroundColor" in JS // Hyphens become camelCase!
Style me with JavaScript!

5. 🎯 Practice Exercises

Try these in your console (F12):

  1. Type document.querySelector('h1') and see what you get.
  2. Change this page's title: document.querySelector('h1').textContent = "My Page"
  3. Change a section's background: document.querySelector('.section').style.backgroundColor = 'lightyellow'
  4. Count all buttons: document.querySelectorAll('button').length
💡 Key Takeaway:
querySelector + textContent + style — these three let you find, read, and change anything on the page!
