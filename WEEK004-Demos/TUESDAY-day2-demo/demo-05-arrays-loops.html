    2. Rendering Arrays to the Page

    const students = ['Alice', 'Bob', 'Charlie']; const list = document.querySelector('#studentList'); // Clear the list first list.innerHTML = ''; // Loop through array and create HTML for each item students.forEach(function(student, index) { const li = document.createElement('li'); li.textContent = `${index + 1}. ${student}`; list.appendChild(li); }); // This is the KEY PATTERN for displaying data on a page!

      ✅ The Render Pattern:
      1. Keep data in an array (your "source of truth")
      2. Create a renderList() function
      3. Clear the list element: list.innerHTML = ''
      4. Loop through array, create elements for each item
      5. Call renderList() after every data change

      3. Loop Types Comparison

      const colors = ['red', 'green', 'blue']; // 1. for loop (classic) for (let i = 0; i < colors.length; i++) { console.log(colors[i]); } // 2. for...of (modern, cleaner) for (const color of colors) { console.log(color); } // 3. forEach (array method — most common for rendering) colors.forEach(function(color, index) { console.log(`${index}: ${color}`); }); // 4. forEach with arrow function colors.forEach((color, i) => console.log(`${i}: ${color}`));

      4. Useful Array Methods

      const numbers = [5, 3, 8, 1, 9, 2]; // filter — keep items that pass a test const big = numbers.filter(n => n > 5); // [8, 9] // map — transform each item const doubled = numbers.map(n => n * 2); // [10, 6, 16, 2, 18, 4] // find — get first item that passes test const found = numbers.find(n => n > 5); // 8 // sort — sort the array const sorted = [...numbers].sort((a, b) => a - b); // [1, 2, 3, 5, 8, 9]
