🎯 Demo 03: Functions with the DOM

Writing functions that read user input and update the page

1. Functions That Read Input

// HTML: <input id="nameInput"><button onclick="sayHello()"> function sayHello() { const name = document.querySelector('#nameInput').value; const result = document.querySelector('#result'); result.textContent = `Hello, ${name}!`; }

✅ Key Pattern:
.value gets the text typed into an input field. This is different from .textContent which gets the text inside a regular element (like a paragraph).

2. Functions with Calculations

function calculateAge() { const birthYear = Number(document.querySelector('#yearInput').value); const currentYear = new Date().getFullYear(); const age = currentYear - birthYear; document.querySelector('#ageResult').textContent = `You are ${age} years old!`; } // ⚠️ Number() converts string from input to a number // Without it: "2025" - "2000" would still work (JS coercion) // But "2025" + "2000" would give "20252000" (string concat!)

⚠️ Always convert input to Number!
Input values are always strings. Use Number() or parseInt() before doing math.
Otherwise: "5" + "3" = "53" (string concatenation, not 8!)

3. Functions That Return Values

// Reusable function — returns a value, doesn't touch the DOM function calculateBMI(weight, height) { return weight / (height * height); } // Separate function to handle the UI function showBMI() { const weight = Number(document.querySelector('#weight').value); const height = Number(document.querySelector('#height').value); const bmi = calculateBMI(weight, height); const category = bmi < 18.5 ? "Underweight" : bmi < 25 ? "Normal" : bmi < 30 ? "Overweight" : "Obese"; document.querySelector('#bmiResult').textContent = `BMI: ${bmi.toFixed(1)} — ${category}`; }

✅ Best Practice — Separate Logic from UI:
  • Logic function: Does the calculation, returns a value
  • UI function: Reads inputs, calls logic, shows result
  • This makes your code reusable and easier to test!

4. Arrow Functions as Event Handlers

// Arrow function syntax const greet = (name) => `Hello, ${name}!`; // Arrow function as event handler const btn = document.querySelector('#myBtn'); btn.addEventListener('click', () => { const name = document.querySelector('#input').value; document.querySelector('#output').textContent = greet(name); });
