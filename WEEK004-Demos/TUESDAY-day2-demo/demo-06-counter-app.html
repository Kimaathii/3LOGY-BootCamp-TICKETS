How It's Built — Step by Step

1 HTML Structure

<!-- The display and buttons --> <div id="counter-display">0</div> <button id="btn-decrement">Decrease</button> <button id="btn-reset">Reset</button> <button id="btn-increment">Increase</button>

2 JavaScript Variables (State)

// Store the counter value in a variable let count = 0; let step = 1; // Get references to HTML elements const display = document.querySelector('#counter-display');

3 Update Function (renders state to DOM)

function updateDisplay() { display.textContent = count; // Change color based on value if (count > 0) { display.style.color = '#2ecc71'; // Green for positive } else if (count < 0) { display.style.color = '#e74c3c'; // Red for negative } else { display.style.color = 'white'; // White for zero } }

4 Event Listeners (connect buttons to functions)

// Increment button document.querySelector('#btn-increment').addEventListener('click', () => { count += step; // Change the data updateDisplay(); // Update the UI }); // Decrement button document.querySelector('#btn-decrement').addEventListener('click', () => { count -= step; updateDisplay(); }); // Reset button document.querySelector('#btn-reset').addEventListener('click', () => { count = 0; updateDisplay(); });

📐 The Pattern You'll Use Everywhere

// THE PATTERN (memorize this!): // 1. DATA — Store state in variables let data = initialValue; // 2. RENDER — Function to update the UI from data function render() { element.textContent = data; } // 3. EVENTS — User actions modify data, then re-render button.addEventListener('click', () => { data = newValue; // Change data render(); // Update UI }); // This is called the "State → Render" pattern // React.js is built entirely on this concept!
✅ This Pattern Powers Everything:
  • Counter app: count variable → display number
  • To-Do list: tasks array → render list items
  • Shopping cart: items array → render product cards
  • React.js: useState → component re-render (same concept!)
