🎯 Color Tips

  • Use hex (#) for most cases - short and clear
  • Use RGBA for transparency needs
  • Use CSS variables for easy theming
  • Visit coolors.co for palettes
  • Check contrast with DevTools (4.5:1 minimum)
  • Limit to 3-5 colors per design
