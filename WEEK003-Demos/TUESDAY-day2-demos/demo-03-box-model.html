🎁 The CSS Box Model

Visual Layers (from outside in)

MARGIN (transparent, pushes elements away)
BORDER (visible edge)
PADDING (inside space, background shows)
CONTENT (text, images)

⚠️ The Problem vs ✅ The Solution

Default (content-box)

width: 200px
Actual width: 250px!

200 + (20×2) + (5×2) = 250px

Problem: Width grows with padding/border!

With border-box

width: 200px
Actual width: 200px!

Padding/border included in width

Solution: Width means TOTAL width!

👉 Inspect both boxes in DevTools to see the box model diagram!

🎯 Best Practice

Add this to EVERY CSS file:

* {
  box-sizing: border-box;
}

This makes ALL elements use border-box. CSS becomes predictable!
