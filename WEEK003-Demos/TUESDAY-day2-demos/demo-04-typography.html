Typography & Text Styling

Font Families

Serif fonts (Georgia) - traditional, readable for long text

Sans-serif fonts (Arial) - modern, clean, web-friendly

Monospace fonts (Courier) - code, fixed-width characters

font-family: Arial, Helvetica, sans-serif;

Font Sizes

Small text (14px)

Medium text (18px) - good for body

Large text (24px) - good for headings

font-size: 16px; /* or em, rem, % */

Font Weight (Thickness)

Light text (300)

Normal text (400)

Bold text (700)

font-weight: bold; /* or 100-900 */

Text Alignment

Left aligned (default)

Center aligned

Right aligned

text-align: center;

Text Transform

Uppercase text

LOWERCASE TEXT

capitalize each word

text-transform: uppercase;

Line Height (Spacing Between Lines)

Tight line height (1.2) makes text feel cramped. Lorem ipsum dolor sit amet consectetur adipisicing elit. Quod, nemo.

Normal line height (1.6) is readable and comfortable. Lorem ipsum dolor sit amet consectetur adipisicing elit. Quod, nemo.

Loose line height (2.0) gives lots of breathing room. Lorem ipsum dolor sit amet consectetur adipisicing elit. Quod, nemo.

line-height: 1.6; /* 1.5-1.6 is ideal */

✅ Typography Best Practices

  • Body text: 16px minimum
  • Line height: 1.5-1.6 for readability
  • Line length: 50-75 characters
  • Contrast: 4.5:1 for normal text, 3:1 for large text
  • Use web-safe fonts or Google Fonts
