Responsive Images & Typography

Learning to make content adapt to any screen size

📐 Responsive Typography

✨ Three Approaches to Responsive Text:

  1. Media Queries: Set different font sizes at breakpoints (traditional)
  2. Viewport Units: Use vw (viewport width) for fluid sizing
  3. clamp(): Set min, preferred, and max sizes (modern best practice)
/* Approach 1: Media Queries */ h1 { font-size: 2rem; /* Mobile */ } @media (min-width: 768px) { h1 { font-size: 3rem; /* Tablet */ } } @media (min-width: 1024px) { h1 { font-size: 3.5rem; /* Desktop */ } } /* Approach 2: Viewport Units */ h1 { font-size: 5vw; /* 5% of viewport width */ } /* Approach 3: clamp() - BEST! */ h1 { /* min, preferred (fluid), max */ font-size: clamp(2rem, 5vw, 4rem); }
⚠️ Accessibility Warning: Never use font sizes smaller than 16px for body text. Users with vision impairments need readable text!

🖼️ Responsive Images

Technique 1: Max-Width 100% (Most Common)

Image scales down on smaller screens but never larger than its original size:

📷 Demo Image
.responsive-img { width: 100%; /* Fill container width */ max-width: 100%; /* Don't exceed original size */ height: auto; /* Maintain aspect ratio */ }

Technique 2: Object-Fit (Fixed Aspect Ratio)

When you need a specific height but want the image to crop nicely:

🎨 Fixed Height Image
.fixed-ratio-img { width: 100%; height: 300px; /* Fixed height */ object-fit: cover; /* Crop to fill */ object-position: center; /* Center the crop */ } /* object-fit options: - cover: crops to fill (most common) - contain: shows entire image (may have empty space) - fill: stretches to fit (distorts image) */

Technique 3: Background Images

Use CSS background for decorative images:

🌄 Background Image
.bg-image { background-image: url('image.jpg'); background-size: cover; /* Fill container */ background-position: center; /* Center image */ background-repeat: no-repeat; height: 400px; }

✅ Best Practices Checklist

Images:

  • ✓ Always include alt text for accessibility
  • ✓ Use width: 100%; height: auto; for flexible images
  • ✓ Optimize images before uploading (compress, resize)
  • ✓ Use appropriate formats (WebP for photos, SVG for icons)
  • ✓ Consider lazy loading for below-the-fold images
  • ✓ Use srcset for art direction (different images at different sizes)

Typography:

  • ✓ Minimum 16px for body text (accessibility)
  • ✓ Use relative units (rem, em) instead of px when possible
  • ✓ Limit line length to 50-75 characters for readability
  • ✓ Increase line-height for better readability (1.5-1.8)
  • ✓ Use system fonts or web fonts with fallbacks
  • ✓ Test with browser zoom (users may increase text size)

🔍 Advanced: srcset for Different Screen Sizes

<!-- Serve different image sizes based on screen width --> <img src="image-800.jpg" srcset=" image-400.jpg 400w, image-800.jpg 800w, image-1200.jpg 1200w " sizes=" (max-width: 600px) 400px, (max-width: 900px) 800px, 1200px " alt="Responsive image example" > <!-- Browser automatically picks the best size! -->
📚 Note: srcset and sizes are advanced. Focus on width: 100%; height: auto; first. We'll cover this in detail in Week 4!
