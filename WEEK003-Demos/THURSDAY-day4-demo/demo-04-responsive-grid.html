📐 How This Grid Works

This uses CSS Grid with mobile-first responsive design. The number of columns changes based on screen width:

  • 📱 Mobile (<600px): 1 column
  • 📱 Tablet (600-899px): 2 columns
  • 💻 Desktop (900-1199px): 3 columns
  • 🖥️ Large (1200-1599px): 4 columns
  • 🖥️ Extra Large (1600px+): 5 columns
💡 Pro Tip: CSS Grid is perfect for card layouts because it automatically handles alignment and spacing, even when cards have different heights!
/* Mobile-first grid */ .card-grid { display: grid; gap: 20px; grid-template-columns: 1fr; /* 1 column */ } /* Tablet and up */ @media (min-width: 600px) { .card-grid { grid-template-columns: repeat(2, 1fr); /* 2 columns */ } } /* Desktop and up */ @media (min-width: 900px) { .card-grid { grid-template-columns: repeat(3, 1fr); /* 3 columns */ } }
