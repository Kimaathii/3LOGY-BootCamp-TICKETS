🎯 flex-wrap & gap - Responsive Layouts

Part 1: flex-wrap (Multi-line Layouts)

By default, flex items try to fit on a single line, even if they overflow the container.
flex-wrap controls whether items can wrap to new lines.

flex-wrap: nowrap; (default) - Items stay on one line

flex-wrap: wrap; is ESSENTIAL for responsive designs!
Without it, items shrink or overflow on small screens.
With it, items automatically move to new lines = responsive magic!

Part 3: Why gap is Better than Margin Hacks

❌ OLD WAY: Margin Hacks
1
2
3
.item {
  margin-right: 20px;
}

.item:last-child {
  margin-right: 0;  /* Annoying! */
}

Problems:

  • Need :last-child hack
  • Breaks with wrapping
  • More CSS to write
  • Hard to modify
✅ NEW WAY: gap Property
1
2
3
.container {
  display: flex;
  gap: 20px;  /* Done! */
}

Benefits:

  • One line of CSS!
  • Works with wrapping
  • No :last-child needed
  • Easy to change
🎯 Rule: ALWAYS use gap for spacing flex items. Never use margin hacks!

Part 4: Responsive Card Grid (wrap + gap)

Combining flex-wrap: wrap; + gap creates responsive layouts that automatically adapt to screen size!

This is how modern responsive grids are built!

.card-grid {
  display: flex;
  flex-wrap: wrap;     /* Allow wrapping */
  gap: 1rem;           /* 16px spacing */
}

.card {
  flex: 1 1 200px;  /* Grow, shrink, minimum 200px */
  padding: 1.5rem;
  background-color: white;
  border-radius: 8px;
}
🎓 Real-World Usage:
This pattern is used for:
- Product grids (e-commerce)
- Photo galleries
- Blog post cards
- Team member profiles
- Portfolio projects

🎯 Key Takeaways

  • flex-wrap: wrap; allows items to wrap to multiple lines
  • Default nowrap keeps all items on one line (may overflow/shrink)
  • gap is the modern way to add spacing between flex items
  • gap is WAY better than margin hacks (no :last-child needed!)
  • Combining wrap + gap creates responsive layouts automatically
  • Cards with flex: 1 1 200px; + flex-wrap: wrap; = responsive grid!

Photo Gallery

.gallery {
  display: flex;
  flex-wrap: wrap;
  gap: 1rem;
}

.photo {
  flex: 1 1 250px;  /* Min 250px, wraps */

Product Grid

.products {
  display: flex;
  flex-wrap: wrap;
  gap: 2rem;
}

.product-card {
  flex: 1 1 300px;  /* Min 300px */
}

Navigation with Wrapping

nav ul {
  display: flex;
  flex-wrap: wrap;   /* Stack on mobile */
  gap: 1.5rem;
  list-style: none;
