🎯 Flexbox Basics - The Foundation

Example 1: Default Block Behavior (NO Flexbox)

By default, <div> elements are block-level, which means they stack vertically (one on top of the other).
Item 1
Item 2
Item 3
<div class="container-default">
  <div class="item">Item 1</div>
  <div class="item">Item 2</div>
  <div class="item">Item 3</div>
</div>
.container-default {
  border: 3px solid red;
  padding: 10px;
  /* No display: flex - default block behavior */
}

.item {
  padding: 20px;
  margin: 5px;
}
What you see: Items stack vertically (default block behavior).
Why: Divs are block-level elements by default, so each takes up full width and stacks.

Example 2: With Flexbox Enabled ✨

Add display: flex; to the parent container, and BOOM - the items automatically arrange in a horizontal row!
Item 1
Item 2
Item 3
<div class="container-flex">
  <div class="item">Item 1</div>
  <div class="item">Item 2</div>
  <div class="item">Item 3</div>
</div>
.container-flex {
  display: flex;  /* ← THE MAGIC LINE! */
  border: 3px solid green;
  padding: 10px;
}

.item {
  padding: 20px;
  margin: 5px;
}
🎓 Teaching Point:
ONE line of CSS (display: flex;) transforms the layout!
Items that were stacking vertically now align horizontally in a row!

Flexbox Terminology 📚

Flex Container: The parent element with display: flex;
Flex Items: The direct children of the flex container
Main Axis: Primary direction items flow (horizontal by default)
Cross Axis: Perpendicular direction (vertical by default) 
┌─────────────────────────────────────────┐
│  FLEX CONTAINER (parent)                │
│                                         │
│  ┌──────┐  ┌──────┐  ┌──────┐          │
│  │Item 1│  │Item 2│  │Item 3│  ← FLEX ITEMS (children)
│  └──────┘  └──────┘  └──────┘          │
│                                         │
│  ←────── MAIN AXIS ──────→              │
│     (horizontal by default)             │
│                                         │
│  ↑                                      │
│  │ CROSS AXIS                           │
│  │ (vertical)                           │
│  ↓                                      │
└─────────────────────────────────────────┘

🎯 Key Takeaways

  • display: flex; is the ONE line that enables Flexbox
  • Add it to the parent container, not the items
  • Items automatically change from vertical stack to horizontal row
  • The parent becomes the flex container
  • The children become flex items
  • Default direction is horizontal (main axis)
🎓 For Instructor:
- Ask students: "What do you see without flexbox?"
- Add display: flex live, ask: "What changed?"
- Have students predict before revealing
- Emphasize: ONE line changes everything!
- Show in DevTools: inspect container, see "flex" badge
