🏗️ Common Flexbox Patterns

Real-world layouts you'll use every day

Pattern 1: Navigation Bar

Use case: Logo on left, navigation links on right
Key properties: justify-content: space-between + align-items: center
MyLogo
  • Home
  • About
  • Services
  • Contact
nav {
  display: flex;
  justify-content: space-between;  /* Logo ← → Links */
  align-items: center;             /* Vertical center */
  padding: 1rem 2rem;
}

.nav-links {
  display: flex;
  gap: 2rem;          /* Space between links */
  list-style: none;
}
✅ Seen on: 90% of websites! Amazon, Google, YouTube, GitHub...

Pattern 2: Hero Section (Perfect Centering)

Use case: Landing page hero, modal dialogs, error pages
Key properties: justify-content: center + align-items: center

Welcome to Flexbox

Build amazing layouts with ease

.hero {
  display: flex;
  justify-content: center;  /* Horizontal center */
  align-items: center;      /* Vertical center */
  min-height: 400px;
  background: linear-gradient(...);
}
✅ Seen on: Landing pages, modals, 404 pages, loading screens

Pattern 3: Responsive Card Grid

Use case: Product grids, blog posts, photo galleries
Key properties: flex-wrap: wrap + flex: 1 1 280px

Card 1

These cards automatically wrap to new lines when the screen gets smaller. Try resizing your browser!

Card 2

Each card has a minimum width of 280px, then they wrap. Super responsive without media queries!

Card 3

Hover over cards to see the lift effect. This pattern is used everywhere in modern web design.

Card 4

The flex: 1 1 280px means: grow if space, shrink if needed, minimum 280px width.

Card 5

Gap property keeps consistent spacing between cards on all screen sizes.

Card 6

Professional, responsive, beautiful. This is how modern sites work!

.card-grid {
  display: flex;
  flex-wrap: wrap;   /* Allow wrapping */
  gap: 1.5rem;       /* Spacing between cards */
}

.card {
  flex: 1 1 280px;   /* Grow, shrink, min 280px */
  padding: 1.5rem;
}
✅ Seen on: E-commerce sites, portfolios, image galleries, blog grids

Pattern 4: Sidebar + Main Content

Use case: Admin dashboards, documentation sites, blogs
Key properties: flex: 0 0 200px (sidebar) + flex: 1 (main)

Website Header

Navigation

  • Dashboard
  • Settings
  • Profile
  • Logout

Main Content Area

The sidebar stays fixed at 200px width.

The main area (this section) takes all remaining space.

Resize the browser - sidebar stays 200px, main area grows/shrinks!

Footer always at bottom 
.page-layout {
  display: flex;
  flex-direction: column;
  min-height: 100vh;
}

.page-body {
  display: flex;
  flex: 1;  /* Takes remaining height */
}

.sidebar {
  flex: 0 0 200px;  /* Fixed 200px width */
}

.main-area {
  flex: 1;  /* Takes remaining width */
}
✅ Seen on: Admin panels, documentation (MDN, React docs), WordPress backend

Pattern 5: Holy Grail Layout (Two Sidebars)

Use case: Complex page layouts with left nav + main + right ads
Key properties: Two fixed sidebars + flexible main

Header

Left Nav

150px fixed

Main Content

This area takes all remaining space between the two sidebars.

Called "Holy Grail" because it was nearly impossible with floats!

With Flexbox? Easy!

Right Ads

150px fixed

Footer 
.holy-grail {
  display: flex;
  flex-direction: column;
  min-height: 100vh;
}

.hg-content {
  display: flex;
  flex: 1;
}

.hg-left-sidebar {
  flex: 0 0 150px;  /* Fixed left */
}

.hg-main {
  flex: 1;  /* Flexible middle */
}

.hg-right-sidebar {
  flex: 0 0 150px;  /* Fixed right */
}
✅ Seen on: News sites, forums, older blogs (becoming less common on mobile)

Pattern 6: Media Object

Use case: Comments, notifications, social media posts
Key properties: Fixed image + flexible text
👤

John Doe

This is a media object pattern. Fixed-width image on the left, flexible text content on the right. Used for comments, tweets, Facebook posts, notifications...

🎨

Design Team

Another example! The image stays 100px, the text takes remaining space. Perfect for any image + text combination.

.media-object {
  display: flex;
  gap: 1rem;
  align-items: flex-start;  /* Align to top */
}

.media-image {
  flex: 0 0 100px;  /* Fixed 100px */
}

.media-content {
  flex: 1;  /* Takes remaining space */
}
✅ Seen on: Twitter, Facebook, Reddit comments, notification lists

Pattern 7: Form with Aligned Labels

Use case: Forms with labels and inputs side-by-side
Key properties: Fixed label width + flexible input 
.form-group {
  display: flex;
  gap: 1rem;
  align-items: center;
}

.form-group label {
  flex: 0 0 150px;  /* Fixed label width */
  font-weight: bold;
}

.form-group input {
  flex: 1;  /* Input takes remaining space */
}
✅ Seen on: Contact forms, registration forms, account settings

Pattern 8: Button Group

Use case: Action buttons, tab navigation, toolbars
Key properties: gap for spacing, centered alignment 
.button-group {
  display: flex;
  gap: 1rem;
  justify-content: center;  /* Center buttons */
}

.btn {
  padding: 0.75rem 1.5rem;
  border: none;
  cursor: pointer;
}
✅ Seen on: Forms, modals, toolbars, action panels
