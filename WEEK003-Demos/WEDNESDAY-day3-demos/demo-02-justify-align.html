🎯 justify-content & align-items

Part 1: justify-content (Main Axis Control)

justify-content controls alignment along the MAIN AXIS (horizontal by default).
Think of it as: "How should items be spaced horizontally?"

🎓 When to use each:
flex-start: Default, items start at left
center: Center buttons, logos
space-between: Logo left, nav right (navigation bars!)
space-evenly: Even distribution (button groups, tabs)

Part 2: align-items (Cross Axis Control)

align-items controls alignment along the CROSS AXIS (vertical by default).
Think of it as: "How should items be aligned vertically?"
Note: Container needs height to see vertical alignment!

align-items: stretch; (default)

🎓 When to use each:
stretch: Default, equal-height cards
center: Vertical centering (most common!)
flex-start: Top alignment
baseline: Aligning text of different sizes

Part 3: The Holy Grail - Perfect Centering 🎯

Combine justify-content: center; + align-items: center; for perfect centering both horizontally AND vertically!
✨ Perfectly Centered! ✨ 
.container {
  display: flex;
  justify-content: center;  /* Horizontal center → */
  align-items: center;      /* Vertical center ↕ */
  height: 300px;            /* Need height to see vertical */
}
🎉 This is what developers dreamed of for 15 years!
Before Flexbox, centering required complex hacks with position, transform, margins...
Now? Two lines. Done. Magic!
💡 Real-world use cases:
- Hero sections on landing pages
- Modal dialogs
- Error pages (404)
- Loading spinners
- Login forms

Understanding Main vs Cross Axis

flex-direction: row (default)
┌─────────────────────────────────┐
│                                 │
│  ←──── MAIN AXIS (→) ────→      │
│  justify-content controls this  │
│                                 │
│  [1]     [2]     [3]            │
│  ↕       ↕       ↕              │
│  CROSS  CROSS  CROSS            │
│  AXIS   AXIS   AXIS             │
│  (align-items controls this)    │
│                                 │
└─────────────────────────────────┘
Remember:
justify-content = main axis (horizontal by default)
align-items = cross axis (vertical by default)

Axes switch with flex-direction! (We'll see this in the next demo)

🎯 Key Takeaways

  • justify-content controls horizontal spacing (main axis)
  • align-items controls vertical alignment (cross axis)
  • Use space-between for navigation bars (logo left, links right)
  • Use center + center for perfect centering
  • Container needs height to see vertical alignment
  • Default: justify-content: flex-start, align-items: stretch
