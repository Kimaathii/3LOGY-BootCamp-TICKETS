Accessible Form Patterns

Instructor: Demonstrate label-input connection by clicking labels. Show tab navigation through form.

Example 1: Basic Contact Form

Demonstrates essential accessibility patterns:

Enter your first and last name

We'll never share your email

Format: 123-456-7890 (optional)

Minimum 10 characters

✅ Accessibility Features:

  • Every input has a matching label (for/id)
  • Required fields marked with asterisk and attribute
  • Help text provides guidance
  • Input types provide validation (email, tel)
  • Autocomplete helps fill forms faster
  • Placeholder text shows example format

Example 3: HTML5 Validation

Try: Submit form with invalid data to see browser validation!

Must be a valid email address

Must be between 13 and 120

Must start with http:// or https://

5 digits only (US ZIP code)

3-20 characters

🔍 HTML5 Validation Attributes:

  • required - Field must be filled
  • type="email" - Must contain @
  • type="url" - Must be valid URL
  • min / max - Number range
  • minlength / maxlength - Text length
  • pattern - Regular expression match
  • title - Shows in validation message

Example 4: Best Practices Summary

✅ Always Do:

  • Match label for="" with input id=""
  • Use descriptive label text (not just "Name" but "Full Name")
  • Mark required fields with required attribute AND visual indicator (*)
  • Use appropriate input types (email, tel, url, number, date, etc.)
  • Provide help text for complex fields
  • Group related inputs with <fieldset> and <legend>
  • Test Tab navigation (logical order)
  • Test keyboard-only form submission
  • Use autocomplete attributes to help users

❌ Never Do:

  • Use placeholder as label (placeholders disappear!)
  • Create custom inputs without keyboard support
  • Rely on color alone to show required fields
  • Use disabled when you mean readonly
  • Forget to test with Tab key navigation
  • Hide focus indicators

🎯 Try This: Tab Navigation Demo

Exercise: Navigate through any form above using only keyboard!

  1. Click anywhere on the page first
  2. Press Tab to move forward through fields
  3. Press Shift + Tab to move backward
  4. Use Space to check/uncheck checkboxes
  5. Use arrow keys in radio buttons and select dropdowns
  6. Press Enter to submit form
